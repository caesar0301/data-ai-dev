HOW TO USE:
--------------------------------------------------------------------------------

1. Activate virtual environment:
   source venv/bin/activate

2. Run the working test:
   python3 test_basic_graph.py

3. Start using NeuG for graph operations:
   import neug
   db = neug.Database("/path/to/db")
   db.load_builtin_dataset("tinysnb")
   conn = db.connect()
   result = conn.execute("MATCH (a)-[:knows]->(b) RETURN a.name, b.name")

--------------------------------------------------------------------------------
NEXT STEPS IF YOU NEED EXTENSIONS:
--------------------------------------------------------------------------------

To use JSON/Parquet extensions, build NeuG from source:

cd neug
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=RELEASE -DBUILD_EXECUTABLES=ON -DBUILD_TEST=ON
make -j$(sysctl -n hw.ncpu)

This will compile the extension libraries and make them available.
